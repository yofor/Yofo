# NaijaMarket MVP Roadmap (6 weeks)

## Week 1 — Product + foundation
- Finalize PRD and scope freeze.
- Initialize monorepo/app scaffold.
- Set up auth, user profiles, and basic UI shell.

## Week 2 — Listings + media proof
- Implement listing create/read flows.
- Implement signed uploads for images and proof videos.
- Add proof validation field for dynamic code.

## Week 3 — Chat + negotiation
- Build conversation threads tied to listing.
- Realtime messaging with persisted message history.
- Add moderation checks for obvious off-platform payment language.

## Week 4 — Orders + escrow lifecycle
- Create order flow and state machine.
- Integrate payment provider sandbox callbacks.
- Hold/release/refund states persisted.

## Week 5 — Delivery + dispute ops
- Delivery event timeline and manual rider updates.
- Buyer confirm delivery action.
- Dispute opening and admin decision panel.

## Week 6 — Score + launch readiness
- Seller score v1 recalculation and badge tiers.
- QA on core flows and edge cases.
- Prepare beta launch checklist and support playbooks.

## Launch checklist (MVP)
- [ ] Core flow succeeds end-to-end on staging
- [ ] Escrow callbacks tested (hold/release/refund)
- [ ] Dispute decision log complete and auditable
- [ ] Terms, privacy, and policy pages published
- [ ] Incident response channel set up
